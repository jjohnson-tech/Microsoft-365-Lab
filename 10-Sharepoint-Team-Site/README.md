## 10 - SharePoint Team Site Creation

A SharePoint team site is a collaborative workspace in Microsoft 365 where a group of people can create, share, and manage content together. Team sites include a document library, lists, pages, and direct integration with Microsoft Teams. In this lab a private IT Help Desk team site was created to simulate how IT departments provision a dedicated SharePoint workspace for their team.

---

Opened a browser and went directly to **sharepoint.com** and signed in using the **javierjohnson@jjtechlab.onmicrosoft.com** account. This takes you straight to the SharePoint home page without going through the Microsoft 365 Copilot portal.

## Step 3: Select Team Site

On the site type selection screen, selected **Team site**. A team site is designed for collaboration within a specific group — it connects to a Microsoft 365 Group and can be linked to a Microsoft Teams channel. This is different from a Communication site which is used for broadcasting information to a wider audience.

## Step 4: Select a Template

Browsed the available templates and selected the **Standard team** template for a general purpose team site. Reviewed the template details and site capabilities, then clicked **Use template** to proceed.

## Step 5: Configure Site Details

Filled in the site details:

- **Site name:** IT Help Desk
- **Site email address:** https://jjtechlab.sharepoint.com/sites/ithelpdesk2 (auto-generated based on the name)
- **Site description:** Private SharePoint workspace for IT Help Desk staff — used for documentation, resources, and team collaboration
- **Privacy settings:** Private — only members can access this site

Setting the site to Private ensures only authorized IT staff can access the workspace. Anyone outside the membership list will not be able to find or access the site.

## Summary

| Task | Action |
|---|---|
| Navigate to SharePoint | Went directly to sharepoint.com and signed in |
| Click + Create Site | Used the Create site button on the SharePoint home page |
| Select Team Site | Chose Team site type for group collaboration |
| Select Template | Used the Standard team template |
| Configure Details | Set site name, description, and privacy to Private |
| Set Language | Confirmed English as the default site language |
| Add Members | Added owner and member during provisioning |
| Confirm Site | Verified site home page loaded correctly after creation |

SharePoint team sites are one of the core collaboration tools in Microsoft 365. Provisioning a site with the correct privacy settings, template, and membership is a common task for IT administrators and helpdesk teams — especially during onboarding new departments or project teams.
