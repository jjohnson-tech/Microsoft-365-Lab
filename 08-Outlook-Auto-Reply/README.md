## 08 - Outlook Auto Reply

An automatic reply (also known as an out of office message) is an email response that is automatically sent to anyone who emails a user during a specified time period. This is commonly used when an employee is on vacation, out sick, or unavailable for an extended period. Configuring auto replies is a frequent helpdesk request. These steps reflect the current new Outlook interface where the File menu has been removed and account settings now live inside View Settings.

---

## Step 1: Open Outlook and Access View Settings

Logged into Microsoft Outlook at **outlook.office.com** using the Javier Johnson account. In the new Outlook, the File menu no longer exists. Instead, clicked **View Settings** in the top right area of the Outlook window to access account-level settings.

## Step 2: Navigate to Accounts → Automatic Replies

Inside View Settings, navigated to **Accounts → Automatic Replies**. This is where Microsoft now centralizes all automatic reply and out of office controls in the new Outlook experience.

## Step 3: Turn On Automatic Replies

Toggled **Turn on automatic replies** to On. At this stage Outlook does not begin sending replies yet — the toggle simply unlocks the configuration options below, giving time to review all settings before activating them.

## Step 4: Enable Send Replies Only During a Time Period

Checked the **Send replies only during a time period** option and configured the start and end dates and times:

- **Start time:** Monday 8:00 AM
- **End time:** Friday 5:00 PM

Setting a scheduled time range ensures the auto reply only sends during the intended period and turns off automatically when the user returns — no manual intervention needed after setup.

<img width="1827" height="845" alt="automatic replies turned on" src="https://github.com/user-attachments/assets/ae04b71f-8cfb-4120-a92e-568c573ddb81" />

## Step 5: Write the Automatic Reply Message

Typed a custom automatic reply message in the **Send automatic replies inside your organization** field:

> Thank you for your email. I am currently out of the office and will return on Monday. If you need immediate assistance please contact the IT Help Desk at helpdesk@jjtechlab.onmicrosoft.com.

Optionally checked **Send replies outside your organization** and added a similar message for external senders.

A clear auto reply message sets expectations for the sender and redirects urgent requests to the appropriate contact so nothing falls through the cracks while the user is away.

## Step 6: Save the Settings

Clicked **Save** to apply the automatic reply configuration. The settings take effect immediately at the configured start time. Closing the settings panel without clicking Save discards all changes.

## Summary

| Task | Action |
|---|---|
| Open View Settings | Accessed account settings via View Settings in new Outlook |
| Navigate to Automatic Replies | Found auto reply under Accounts → Automatic Replies |
| Turn On Auto Reply | Toggled automatic replies on to unlock configuration options |
| Set Time Period | Configured start and end date and time for the reply window |
| Write Message | Added a custom out of office message with alternate contact |
| Save Settings | Clicked Save to activate the automatic reply configuration |

In the new Outlook, Automatic Replies moved from the old File menu to View Settings → Accounts → Automatic Replies. Knowing this updated path is important for helpdesk staff assisting users who can no longer find the feature where they expect it.
