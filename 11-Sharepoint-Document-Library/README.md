## 11 - SharePoint Document Library

A document library in SharePoint is a secure location where files can be stored, organized, shared, and accessed by team members. Every SharePoint team site comes with a default document library, but additional folders and libraries can be created to keep content organized by category or purpose. In this lab a Help Desk Resources folder was created inside the IT Help Desk site's document library to organize IT resources and simulate real-world file management tasks.

---

Opened a browser and went to **sharepoint.com**, signed in using the **javierjohnson@jjtechlab.onmicrosoft.com** account, and navigated to the **IT Help Desk** team site created in the previous section.

## Step 3: Create a New Folder

Inside the Documents library, clicked **+ New** at the top of the page and selected **Folder** from the dropdown menu to create a new folder inside the library.

## Step 4: Name the Folder

In the folder creation dialog, typed the folder name:

- **Folder name:** Help Desk Resources

Clicked **Create** to add the folder to the document library. This folder will serve as the central location for IT Help Desk documentation, guides, templates, and reference materials.

Clicked **+ New → File upload** to upload a file into the Help Desk Resources folder, simulating how IT staff would add documentation or resources to the shared workspace.

## Summary

| Navigate to Site | Opened the IT Help Desk SharePoint team site |
| Open Documents Library | Clicked Documents in the left navigation panel |

| Confirm Folder | Verified the folder appeared in the document library |
| Upload File | Uploaded a file into the folder to simulate real-world use |

Document libraries and folder organization are core SharePoint skills for helpdesk and IT support teams. Knowing how to create folders, upload files, and manage content inside a SharePoint library is a task that comes up frequently when provisioning team workspaces or supporting end users with file management requests.
